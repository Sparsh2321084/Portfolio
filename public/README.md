# Image Folders

## /public/projects/
Place your project images here:
- Project main images (e.g., `iot-system.jpg`)
- Schematics (e.g., `iot-schematic.png`)
- Demo videos (e.g., `iot-demo.mp4`)

## /public/gallery/
Place your gallery images here:
- PCB photos (e.g., `pcb-top.jpg`, `pcb-bottom.jpg`)
- Schematics (e.g., `schematic-1.png`)
- Debug traces (e.g., `logic-trace.png`)
- Assembly photos (e.g., `robot-assembly.jpg`)
- CAD renders (e.g., `enclosure.jpg`)

## /public/
Place your resume here:
- `resume.pdf`

## Tips:
- Keep filenames lowercase with hyphens
- Compress images before uploading (use TinyPNG.com)
- Recommended sizes:
  - Project images: 1200x800px
  - Gallery images: 800x600px
